# 🏛️ Political Corruption — الفساد السياسي

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **الفساد السياسي في المجتمعات العربية والإسلامية (Political Corruption in Arab and Islamic Societies)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 cards on corruption roots, accountability, justice, Shura, and reform
- 6 reform solutions with problem/solution format
- Daily habit tracker with streaks, self-assessment quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 unique themes (Steel, Night, Justice) with localStorage persistence
- Narrator with karaoke, voice/speed/pitch, sleep timer
- Floating particles background, glassmorphism panels
- Swipe navigation, sound effects, responsive design
- PWA-ready with manifest.json

## Tech

Pure HTML + CSS + JS. No frameworks, no dependencies. Just open `index.html`.
